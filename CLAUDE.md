# Auto Publishing

## Telegram 알림 봇

파이프라인 실행 결과를 텔레그램으로 알림 발송하는 용도로 사용한다.

- **봇**: `@BotFather` 에서 본인이 생성한 봇 (토큰은 `.env` 의 `TELEGRAM_BOT_TOKEN` 으로 주입)
- **용도**: 파이프라인 발행 성공/실패 알림, 스케줄러 상태 알림, 에러 즉시 알림
- **알림 모듈**: `common/notifier.py` — 모든 파이프라인이 완료 시 자동 호출
- **환경변수**: `TELEGRAM_BOT_TOKEN`, `TELEGRAM_CHAT_ID` (.env)
- **텔레그램은 콘텐츠 발행 플랫폼이 아님** — 알림 전용으로만 사용

### 알림 발송 시점
- 모든 파이프라인 `run()` 완료 시 결과 알림 (성공/부분성공/실패)
- `_safe_call` 에서 예외 발생 시 에러 알림
- 스케줄러 시작 시 등록된 작업 수 알림
